Index

Index(rowopt, col)

Represents a 2D index. Can be used as a thread/block index or index into an array. Row-major order is assumed

Constructor

# new Index(rowopt, col)

  • When two arguments are passed the first one defines the row (y) and the second one the column (x).
  • When one argument is passed it defines the column (x). This is to be consistent with common, C-like, row-major indexing notation
Parameters:
Name Type Attributes Description
row Number <optional>

Value for the row, (y) dimension.
col Number

Value for the col (x) dimension.

View Source renderer/models/Index.js, line 107

Examples
let index1 = new Index(1,2) // 3rd element of the 2nd row
let index2 = new Index(9,1) // 2nd element of the 10th row
let index3 = new Index(9,0) // 1st element of the 10th row
let index4 = new Index(9)   // 10th element of the 1st row
let index5 = new Index(0,9) // Equivalent to index4
