# hasGlobalThreadIdx(index)

Check if this block contains a thread with a given global index.

If an integer is passed it will be treated as an index with 0 in the y-dimension. For instance

  • hasGlobalThreadIdx(10)
  • hasGlobalThreadIdx(new CuIndex(0,10))
  • hasGlobalThreadIdx(new CuIndex(10)) are equivalent

To check if the block contains a linear index use module:cuda.CuBlock#hasGlobalLinearIndex
