MemoryVisService

memory-vis~MemoryVisService()

The memory visualization service This service can be used to visualize some memory (see module:memory.Memory) The service mentains no context, i.e it will only create and render the memory and return a controller to it. This means that when analysis for a different kernel takes place, the current visualization(s) must be removed explicitely. See module:memory-vis.MemoryVisService#removeAll The service only mentains a list of the currently open visualizations and an API to query them
